# Simple Facebook Clone

A simple social media web application inspired by Facebook, developed in PHP and MySQL. Users can create, edit, delete posts, and follow/unfollow other users.

Features

User Authentication-

Signup and login system for users.

Post Management-

Create Post: Users can write and publish posts.

Edit Post: Users can edit their existing posts.

Delete Post: Users can delete their posts.

Follow/Unfollow Users.
